# O Código do Sono Infantil

Página de vendas e ecossistema do Método do Sono Tranquilo.

## Estrutura do Projeto
- `index.html`: Página de vendas principal (codigodosonoinfantil).
- `hub-temp/`: Página central de links.
- `checklist-temp/`: Material bônus de ambiente de sono.
- `guia-temp/`: Guia de técnicas de relaxamento.
- `planejador-temp/`: Planejador de rotina.

## Tecnologias
- HTML5 / CSS3 (Vanilla)
- Meta Pixel (Facebook) para rastreamento de conversões.
